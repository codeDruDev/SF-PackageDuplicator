# Package Duplicator

Esta extensión permite duplicar packages y renombrarlos con el formato CRMLEAD000Y.

## Características

- Duplica archivos package
- Permite seleccionar la carpeta destino
- Nombra automáticamente según el número de ticket

## Uso

1. Abre el archivo que quieres duplicar
2. Presiona `Ctrl+Shift+P` (o `Cmd+Shift+P` en Mac)
3. Busca "Duplicar Package"
4. Ingresa el número de ticket
5. Selecciona la carpeta destino
